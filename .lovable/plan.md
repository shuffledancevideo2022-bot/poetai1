

## Найденные проблемы

### 1. Тарифы: разовые пакеты не отображаются (BUG)
В `src/pages/Pricing.tsx` фильтрация идет по `product_type === "one_time"`, но в базе данных продукты хранятся с типом `"package"`. Из-за этого секция "Разовые пакеты" всегда пуста -- отображаются только подписки.

**Исправление:** Заменить `"one_time"` на `"package"` в строке 36.

### 2. Футер отсутствует на нескольких страницах
- `PaymentSuccess.tsx` -- нет футера
- `PaymentFailed.tsx` -- нет футера  
- `NotFound.tsx` -- нет футера, текст на английском ("Oops! Page not found")

**Исправление:** Добавить компонент `<Footer />` на эти страницы. Перевести NotFound на русский язык.

### 3. Навигационная шапка только на главной странице
Хедер с навигацией (Мастерская поэта, Библиотека, Блог, Тарифы) реализован только внутри `Index.tsx`. На остальных страницах (Guide, Library, Blog, Pricing) нет единого хедера -- только ссылка "Вернуться на главную".

**Исправление:** Вынести хедер в отдельный компонент `Header.tsx` и подключить его на всех публичных страницах.

### Итого изменений
1. Исправить фильтр `"one_time"` -> `"package"` в Pricing.tsx
2. Создать общий компонент `Header.tsx`
3. Добавить Header и Footer на все страницы (PaymentSuccess, PaymentFailed, NotFound)
4. Перевести NotFound на русский
